# Orion 

Orion is a mini renderer that will be extended to a mini game engine probably in the future.

## Build Example
Dependencies:
- CMake
- Visual Studio 2019+ (If you are on Windows)

Linux

```bash
#if you don't have cmake already
sudo apt install cmake
#in Orion directory
mkdir build
cd build
cmake ..
make
../Orion
```
Windows
```
mkdir build
cd build
cmake ../
Orion.sln
```
